## ⚙️ Cara Pakai

1. Import file `.json` ke n8n:

   * Buka n8n
   * Klik **Import Workflow**
   * Upload file

2. Setup credentials:

   * API Key (jika ada)
   * Telegram Bot Token (jika pakai Telegram)
   * dll

3. Jalankan workflow 🚀

## 💡 Notes

* Pastikan semua credential sudah di-setup sebelum run
